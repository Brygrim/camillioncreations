# Image Naming Convention

## Structure

```
{category}-{type}-{number}.jpg
```

## Examples

| Filename | Meaning |
|----------|---------|
| `chalk-wide-001.jpg` | Chalk art, wide aspect (400×300) |
| `chalk-tall-002.jpg` | Chalk art, tall aspect (300×400) |
| `chalk-square-003.jpg` | Chalk art, square (320×320) |
| `portrait-001.jpg` | Bio portrait image |
| `collage-1.jpg` | Hero slideshow image |

## Categories

- `chalk/` — Chalk art gallery (14 images)
- `portrait/` — Bio headshots (2 images)
- `collage/` — Landing page slideshow (4 images)
- `watercolor/` — Watercolor gallery (awaiting images)
- `ceramics/` — Ceramics gallery (awaiting images)

## Aspect Types

| Type | Dimensions | Use For |
|------|------------|---------|
| `wide` | 400×300 | Landscape artwork |
| `tall` | 300×400 | Portrait artwork |
| `square` | 320×320 | Square/centered |
| (none) | varies | Special cases |

## Rules

- All lowercase
- Use hyphens `-`
- Zero-pad numbers: `001`, `002`
- Always `.jpg` (never `.jpeg`)
