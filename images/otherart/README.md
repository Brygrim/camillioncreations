# Murals Gallery Images

Place mural images in this folder with these naming conventions:

- mural-wide-001.jpg, mural-wide-002.jpg (landscape images)
- mural-tall-001.jpg, mural-tall-002.jpg (portrait images)
- mural-square-001.jpg, mural-square-002.jpg (square images)

Images will appear in the Murals gallery section.
